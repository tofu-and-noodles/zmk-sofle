# AGENTS.md

## Workflow

This is a personal ZMK keymap for rapid prototyping. It is very much a personal,
frequently-changing config, not a shared production repo.

**Commit straight to `main`.** Do not create worktrees, feature branches, or PRs
for changes here. Just make the edit on the current branch and commit directly.
This overrides any global worktree/branch-isolation defaults.
